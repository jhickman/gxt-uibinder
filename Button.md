# Button #

The `Button` class supports nested menus.  This can be done using the following XML:

```
<button:Button text="Click Me">
  <menu:Menu>
    <menu:MenuItem text="Menu Item 1" />
    <menu:MenuItem text="Menu Item 2" />
    <menu:MenuItem text="Menu Item 3" />
  </menu:Menu>
</button:Button>
```
